# Project Context

业务上下文包，供 AI 智能体和开发工具按需读取。

**不要一次性加载本目录下的全部文件。** 先读 `business/INDEX.md`，根据当前任务按需选取 1-3 个文件。

```
project_context/
├── README.md              ← 你正在读的文件
└── business/              ← 业务上下文
    ├── INDEX.md           ← 快速索引（入口）
    ├── manifest.json      ← 结构化条目清单
    ├── flows/             ← 流程说明
    ├── reports/           ← 报表说明
    ├── terms/             ← 术语说明
    └── templates/         ← 模板（新建条目时参考）
```
