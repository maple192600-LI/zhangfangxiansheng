# 资金日报（示例）

> 这是一个示例条目，展示 report 类型文件的写法。业务人员后续可在前端创建正式报表条目。

## 基本信息

| 字段 | 内容 |
|------|------|
| 报表名称 | 资金日报 |
| 所属模块 | 报表查询 |
| 解决什么问题 | 看每个法人每天的资金变动：期初多少钱、收了多少、支了多少、期末剩多少 |

## 数据关系

| 字段 | 内容 |
|------|------|
| 上游数据来自哪里 | FundEvent（只统计 state="正常"的记录）+ Account + Entity |
| 主要展示什么 | 按法人汇总：期初余额、期间收入、期间支出、净变动、期末余额 |
| 会被谁使用 | 财务会计每日查看资金状况 |
| 导出用途 | 发给老板、存档、与其他系统对接 |

## 关联流程

| 字段 | 内容 |
|------|------|
| 关联流程 | 依赖"手工快速录入"或"银行流水导入"提供数据 |

## 给 Agent 的提示

- service 函数：`report_service.daily_report`
- 只统计 `include_in_daily_report=True` 的账户
- 期初 = initial_balance + 历史净变动
- 有模板时按模板列展示，无模板用默认列
- 导出文件名：`资金日报_开始日期_结束日期.xlsx`
