# 20_数据模型与字段字典

## READ WHEN

设计或修改数据库表、字段、主数据、动态业务对象、导入映射、报表字段、工作流字段引用时读取。

## READ ALSO

`../docs/17_数据模型与字段字典设计规范.md`

## MUST

- 主数据使用稳定 `id`。
- 字段定义进入字段字典。
- 页面、报表、工作流、规则、Skill 引用字段 `key` 或 `id`，不引用展示名。
- 名称修改后相关页面必须自动展示调整后的名称。
- 归属修改后相关页面同步或提示影响范围。
- 需要保留口径时保存快照字段。

## NEVER

- 不要用名称做关联键。
- 不要把展示名写死在业务逻辑里。
- 不要每个页面维护一份主数据名称。
- 不要只让设置页生效。
- 不要把快照口径和当前主数据混用。

## DONE

```text
稳定 id：
字段字典：
关联方式：
影响检查：
浏览器验收：
```
