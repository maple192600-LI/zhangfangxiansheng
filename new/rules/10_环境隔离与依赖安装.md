# 10_环境隔离与依赖安装

## READ WHEN

安装依赖、创建虚拟环境、初始化数据库、配置前端、启动服务、打包时读取。

## READ ALSO

`../docs/04_工程环境与依赖隔离规范.md`

## MUST

- Python 使用项目内 `.venv`。
- Node 依赖在 `app/frontend/node_modules`。
- 数据库、日志、缓存放 `data/`。
- 依赖版本写入清单或锁定文件。
- 全局安装或系统环境修改必须先问用户。

## NEVER

- 不要安装到系统 Python。
- 不要把本机依赖目录提交到 GitHub。
- 不要让开发数据库和用户运行数据库混用。
- 不要要求用户手工改系统环境才能运行产品。

## CHECK

- `.venv/` 未提交。
- `node_modules/` 未提交。
- `data/` 未提交。
- 依赖清单存在。
- 本机全局环境未被悄悄修改。
