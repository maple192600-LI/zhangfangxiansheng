# 18_浏览器验收记录

## READ WHEN

实现用户可见功能、修复页面问题、上传文件、导出文件、删除入口、提交 PR 前读取。

## READ ALSO

`../docs/15_浏览器验收记录规范.md`

## MUST

- Claude Code 使用 `web-access skill` 真实操作页面。
- Codex 使用 `browser-use` 或当前环境明确提供的等价浏览器操作 skill。
- 记录地址、工具、样本文件、步骤、预期、实际、刷新后验证、错误场景。
- 验收记录写入 `dev/reports/browser_acceptance/`。
- 下载、截图、视频按临时产物规则处理。
- 完成声明给出用户复查路径。

## NEVER

- 不要只跑后端测试。
- 不要只打开首页。
- 不要只说“已测试”。
- 不要用 Playwright 结果替代 `web-access skill` / `browser-use` 操作记录。
- 不要跳过错误场景。
- 不要用假截图代替真实操作。

## DONE

```text
浏览器验收记录：
结论：
用户复查路径：
未通过项：
```
