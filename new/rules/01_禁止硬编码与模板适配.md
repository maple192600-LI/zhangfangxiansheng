# 01_禁止硬编码与模板适配

## READ WHEN

任务涉及 Excel、银行流水、报表模板、解析器、字段映射、模板学习、规则修正时必须读取。

MUST:

- 代码只提供通用解析能力。
- 模板差异进入规则、映射或工作流配方。
- 每个规则必须有版本、适用条件、样本、试跑结果。
- 未知模板进入学习流程。
- 已知模板变化时生成规则后续版本，不覆盖原版本。

NEVER:

- 把文件名写成判断条件。
- 把固定列号作为唯一逻辑。
- 把固定单元格坐标作为唯一逻辑。
- 把固定 Sheet 名称作为唯一逻辑。
- 为当前样本写 if/else 特判。

## STRUCTURE

```text
通用解析器
  -> 模板指纹
  -> 字段映射
  -> 规则版本
  -> 样本试跑
  -> 用户确认
```

## TEST

- 已知模板：应命中原规则。
- 同类变体：应证明不是单文件硬编码。
- 未知模板：应进入学习流程。
- 格式变化模板：应生成规则修正版。

## CHECK

- 是否存在硬编码文件名、列号、坐标、Sheet 名称？
- 是否有规则版本？
- 是否有样本试跑？
- 是否有用户确认入口？
- 是否能复用到第二个同类文件？
