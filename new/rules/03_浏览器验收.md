# 03_浏览器验收

## READ WHEN

任务涉及任何用户可见页面、按钮、上传、预览、导出、弹窗、错误提示时必须读取。

MUST:

- 使用浏览器工具真实操作页面。
- Claude Code 必须使用 `web-access skill`。
- Codex 必须使用 `browser-use`，或当前环境明确提供的等价浏览器操作 skill。
- Playwright 只能补充自动化或记录，不能替代 `web-access skill` / `browser-use` 的真实操作验收。
- 上传真实样本文件。
- 点击真实按钮。
- 查看真实页面结果。
- 记录操作路径和结果。

NEVER:

- 只跑后端测试就说完成。
- 只看接口返回就说页面完成。
- 用 Playwright 结果替代 Claude Code 的 `web-access skill` 操作记录。
- 用假数据截图冒充真实功能。
- 只打开首页不操作功能。
- 跳过失败路径。

```markdown
## 浏览器验收

- Skill：

- 地址：

  1.
  2.
  3.
- 预期结果：
- 实际结果：
- 截图或记录：
- 结论：
```

## CHECK

- 是否真实打开页面？
- 是否真实点击？
- 是否真实上传或输入？
- 是否页面显示了正确结果？
- 是否验证了错误提示？
- 用户是否能按步骤复查？
