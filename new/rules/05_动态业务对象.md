# 05_动态业务对象

## READ WHEN

任务涉及数据中心、自定义台账、动态字段、字段字典、导入方案、业务对象视图时必须读取。

MUST:

- 业务对象由元数据定义。
- 字段由字段定义驱动。
- 页面由对象定义和视图配置渲染。
- 导入方案绑定到业务对象。
- 权限范围绑定到业务对象。

NEVER:

- 为每种台账写一套硬编码 CRUD。
- 把用户自定义字段写死进前端组件。
- 把字段显示名当数据库字段名。
- 绕过字段字典拼 SQL。

## MIN MODEL

```text
business_object
business_field
business_record
business_view
import_profile
object_permission
```

## CHECK

- 是否能新增对象？
- 是否能新增字段？
- 是否能导入数据？
- 是否能切换视图？
- 是否能通过字段定义渲染表格？
- 是否没有为单一台账写死页面逻辑？
