# 13_开发阶段删除语义

## READ WHEN

开发阶段用户要求删除功能、页面、按钮、接口、模板、测试文件、上传样本、规则草稿、工作流草稿时读取。

## READ ALSO

`../docs/10_开发阶段删除语义规范.md`

## MUST

- 默认按彻底删除处理。
- 清理前端、后端、数据库、测试、样本、文档、工作流、Skill / 规则引用。
- 搜索功能名、路由名、接口路径、组件名、service 名、表名、字段名。
- 残留必须为 0；允许保留的残留必须逐条说明原因。
- 做浏览器验收，确认入口、路由、按钮、列表项都不存在。

## NEVER

- 不要只删前端。
- 不要只隐藏菜单。
- 不要只删列表记录。
- 不要保留后端接口或 service。
- 不要后续从残留内容恢复已删除功能。

## DONE

```text
删除对象：
残留搜索：
残留结果：
浏览器验收：
```
