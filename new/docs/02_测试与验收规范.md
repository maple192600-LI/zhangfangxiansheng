# 02_测试与验收规范

## 0. READ WHEN

任何开发任务进入实现、验证、交付、PR 前必须读取。

---

## 1. TEST LAYERS

| 类型 | 必须回答 |
|---|---|
| 单元测试 | 函数 / 服务逻辑是否正确 |
| 接口测试 | API 请求、响应、错误码是否符合契约 |
| 数据库测试 | 写入、关联、迁移、回滚是否正确 |
| 规则测试 | 模板、映射、解析器是否可复用 |
| 黄金样本测试 | 输入 Excel 与输出 Excel 是否符合预期 |
| 浏览器验收 | 用户真实操作路径是否可用 |
| 回归测试 | 本次改动是否破坏原样本、原规则、原工作流 |

---

## 2. MUST

- 用户可见功能必须做浏览器验收。
- 模板 / 规则 / 工作流必须做样本试跑。
- 涉及主数据必须做跨页面联动验收。
- 涉及字段字典或动态业务对象必须验证字段引用不依赖展示名。
- 涉及 API 必须验证前端真实调用、后端真实生效、错误提示中文可读。
- 涉及导出必须打开或解析导出文件验证内容。
- 涉及错误场景必须验证前端错误提示。
- 涉及数据写入必须验证页面可查询、可追溯。
- 涉及代码替换必须验证原入口、原引用、原文件已删除。
- 涉及开发阶段删除必须验证前端、后端、数据库、测试、文档、工作流引用都已清理。
- 涉及正式产品删除必须验证操作语义、影响范围提示、引用检查和刷新后的状态。
- 涉及临时产物必须验证任务结束前已清理。
- 涉及 GitHub 开发必须验证当前任务不直接在 `main` 开发。
- 完成前必须运行项目护栏检查，或明确写出尚未实现且不得声明完成。
- 完成声明必须给出用户复查路径。

---

## 3. NEVER

- 只跑后端测试就声明完成。
- 只看接口成功就声明页面完成。
- 跳过浏览器验收。
- 用假数据截图代替真实操作。
- 删除失败测试。
- 放宽断言掩盖失败。
- 修改黄金样本掩盖错误输出。
- 只验证成功路径，不验证错误提示。
- 保留失败实现或失败分支。
- 用旁路入口绕过原问题但不删除原入口。
- 开发阶段只删除前端入口但保留后端、数据库、测试或文档残留。
- 正式产品中只删除前端列表项但保留不可见后端残留。
- 正式产品删除不提示影响范围。
- 把临时下载、日志、运行数据库提交到 GitHub。
- 护栏检查失败仍声明完成。

## 6. REQUIRED USER PATHS

### 文件导入

```text
打开文件导入页
上传样本文件
查看识别状态
查看预览表格
查看异常提示
确认或取消导入
进入数据中心查询导入结果
```

### 模板 / 规则

```text
上传已知模板
确认命中规则
上传变体模板
确认未硬编码
上传未知模板
确认进入学习流程
试跑规则草稿
确认保存后续版本
```

### 主数据联动

```text
修改主数据名称或归属
返回相关列表
查看筛选项
查看报表预览
查看导出结果
确认显示同步或出现受影响提示
```

### 报表生成

```text
选择期间 / 参数
生成预览
查看取数来源
查看校验结果
导出 Excel
验证导出内容
查看生成记录
```

### 工作流

```text
创建或生成工作流草稿
查看步骤卡片
样本试跑
查看失败节点或成功结果
用户确认启用
再次执行已启用版本
查看版本记录
```

## 7. REFERENCE SAMPLES

涉及 Excel 输入输出时必须准备：

```text
dev/fixtures/input/
dev/fixtures/expected/
dev/fixtures/variants/
dev/fixtures/unknown/
dev/fixtures/regression/
dev/fixtures/browser/
dev/fixtures/manifest.md
```

样本管理规范见：

```text
16_样本与测试文件管理规范.md
```

NEVER:

- 把 `expected` 改成当前错误输出。
- 只保留一个样本。
- 用样本文件名驱动业务逻辑。
- 把测试样本放进 `app/`。
- 删除样本但不更新 `manifest.md`。

## 8. DONE CHECK

功能完成前逐项确认：

- [ ] 后端自检通过。
- [ ] 浏览器验收通过。
- [ ] 浏览器验收记录已写入，或本任务无用户可见功能。
- [ ] 用户可见页面显示正确。
- [ ] 错误提示可见。
- [ ] 主数据联动已验证，或本任务不涉及。
- [ ] 字段字典和稳定 id 已验证，或本任务不涉及。
- [ ] API 契约和页面 API 映射已验证，或本任务不涉及。
- [ ] 模板硬编码已检查，或本任务不涉及。
- [ ] 导出文件已验证，或本任务不涉及。
- [ ] 规则 / 工作流版本已验证，或本任务不涉及。
- [ ] 原入口和原引用已删除，或本任务不涉及替换。
- [ ] 删除残留搜索为 0，或残留已逐条说明。
- [ ] 产品删除语义和影响范围已验证，或本任务不涉及。
- [ ] 临时文件已清理，或本任务无临时文件。
- [ ] 运行产物未进入 GitHub 提交。
- [ ] 项目护栏检查通过。
- [ ] 用户复查路径已写明。

任一项未完成，不得声明用户功能完成。
