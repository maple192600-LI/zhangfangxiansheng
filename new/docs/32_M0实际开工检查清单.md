## M0 TASK SPLIT

M0 分两个 PR。

### PR 1：仓库初始化与首次提交

目标：

```text
创建私有仓库
建立根目录
迁入已确认文档
放置 skill-creator
创建 index.json
创建 .gitignore
创建占位目录
提交首次 PR
```

不得包含：

```text
产品代码
业务页面
数据库
运行数据
测试样本
业务 Skill
```

### PR 2：项目护栏脚本

目标：

```text
实现 dev/scripts/check_project_guard.py
实现 dev/scripts/check_project_guard.ps1
生成 dev/reports/project_guard_sample_output.md
验证通过样例
验证失败样例
清理失败样例临时文件
```

不得包含：

```text
产品功能
前端页面
后端业务接口
运行数据库
业务样本
```

## SKILL-CREATOR CHECK

`skills/skill-creator/SKILL.md` 必须满足：

```text
存在
是 SKILL.md 兼容结构
可作为全局 Skill 被识别
来源、版本或 hash 写入 skills/index.json
```

`skills/index.json` 最小结构：

```json
{
  "skills": [
    {
      "name": "skill-creator",
      "scope": "global",
      "path": "skills/skill-creator/SKILL.md",
      "enabled": true,
      "status": "enabled",
      "source_kind": "bundled",
      "source_ref": "",
      "version": "",
      "hash": "",
      "allowed_agents": [],
      "permissions": [],
      "dependencies": [],
      "workflow_nodes": [],
      "fixtures": [],
      "test_report_ids": [],
      "created_at": "",
      "updated_at": ""
    }
  ]
}
```

`workspaces/default/skills/index.json` 最小结构：

```json
{
  "skills": []
}
```

## PROJECT GUARD CHECK

PR 2 必须验证：

```text
python dev/scripts/check_project_guard.py
powershell -ExecutionPolicy Bypass -File dev/scripts/check_project_guard.ps1
```

失败样例必须验证：

```text
临时创建一个违规文件
运行护栏脚本
脚本 exit code = 1
输出中文失败原因和 JSON
删除违规文件
再次运行通过
```
