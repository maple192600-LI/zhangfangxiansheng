# 产品范围

## 目标用户

中国财务人员，本地部署，无需命令行操作，双击即可启动。

## V1 包含

- 主数据中心：板块、法人实体、账户、银行信息的 CRUD 和批量管理
- 银行流水导入：上传 Excel → 预览 → 确认提交
- 手工流水双轨：Track A 快速录入 + Track B Excel 上传
- 日报/报表：资金日报、现金日记账、账户余额表、收支明细表等
- 导出打印：Excel 导出 + 打印
- 基础看板：趋势图、构成图
- 备份回滚：数据库备份、恢复、数据清理
- 日志：操作日志、AI 调用日志
- Agent 系统：通用 Agent、会话、工具调用、记忆、技能

## V1 禁止

- 发票 OCR 正式流程
- 费用审批
- 合同管理
- 多人协作
- 集中部署

## 最小闭环边界

- 允许渐进实现：核心功能先可用，增强后补齐
- 不允许：
  - 写死数据或跳过数据库直接返回硬编码结果
  - 跳过用户确认环节
  - 跳过异常处理和状态隔离
  - 跳过真实业务流（如导入必须走 preview → commit）
