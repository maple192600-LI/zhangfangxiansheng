# 数据表关系映射

> 描述核心数据表之间的关系和依赖。

---

## 表分类

### 主数据模块（6 张）

| 表 | 主键 | 关系 | 说明 |
|----|------|------|------|
| `divisions` | id | → entities.division_id | 板块 |
| `entities` | id | → divisions.id, → accounts.entity_id, → fund_events.entity_code | 法人实体 |
| `banks` | id | → accounts.bank_id | 银行 |
| `accounts` | id | → entities.id, → banks.id, → account_aliases.account_id, → fund_events.account_code | 账户 |
| `account_aliases` | id | → accounts.id | 账户别名 |
| `users` | id | — | 用户（单用户认证） |

### 流水事实（2 张）

| 表 | 主键 | 关系 | 说明 |
|----|------|------|------|
| `import_batches` | id | → fund_events.batch_id | 导入批次 |
| `fund_events` | id | → import_batches.id, → entities.entity_code, → accounts.account_code, → parser_artifacts.id | **核心事实表** |

### Agent 产物（3 张）

| 表 | 主键 | 关系 | 说明 |
|----|------|------|------|
| `parser_artifacts` | id | → fund_events.parser_artifact_id | 解析器产物 |
| `rule_artifacts` | id | → report_templates.id, → template_inference_job.rule_artifact_id | 规则产物 |
| `template_inference_job` | id | → rule_artifacts.id | 模板推断任务 |

### 报表（2 张）

| 表 | 主键 | 关系 | 说明 |
|----|------|------|------|
| `report_templates` | id | → rule_artifacts.template_id | 报表模板 |
| `daily_report_runs` | id | — | 日报生成记录 |

### Agent 系统（6 张）

| 表 | 主键 | 关系 | 说明 |
|----|------|------|------|
| `agents_v2` | id | → ai_configs.id, → skills_v2.owner_agent_id, → agent_sessions.agent_id, → agent_memories.agent_id | 智能体 |
| `skills_v2` | id | → agents_v2.id, → agent_runs.skill_id | 技能 |
| `agent_sessions` | id | → agents_v2.id, → agent_messages.session_id, → agent_runs.session_id | 会话 |
| `agent_messages` | id | → agent_sessions.id | 消息 |
| `agent_runs` | id | → skills_v2.id, → agents_v2.id, → agent_sessions.id | 运行记录 |
| `agent_memories` | id | → agents_v2.id | 记忆 |

## 核心关系图（文字版）

```
divisions
  └── entities (1:N, division_id)
        ├── accounts (1:N, entity_id)
        │     ├── account_aliases (1:N, account_id)
        │     └── fund_events (通过 account_code 关联)
        └── fund_events (通过 entity_code 关联)

banks
  └── accounts (1:N, bank_id)

import_batches
  └── fund_events (1:N, batch_id)

parser_artifacts
  └── fund_events (1:N, parser_artifact_id)

report_templates
  └── rule_artifacts (1:N, template_id)

rule_artifacts
  └── template_inference_job (1:N, rule_artifact_id)

ai_configs
  └── agents_v2 (1:N, ai_config_id)

agents_v2
  ├── skills_v2 (1:N, owner_agent_id)
  ├── agent_sessions (1:N, agent_id)
  └── agent_memories (1:N, agent_id)

agent_sessions
  ├── agent_messages (1:N, session_id)
  └── agent_runs (1:N, session_id)
```

## FundEvent 的 CANONICAL_12 字段映射

| 字段 | 数据库列 | 类型 | 约束 |
|------|---------|------|------|
| business_date | `business_date` | Date | NOT NULL |
| entity_code | `entity_code` | String(50) | FK → entities.entity_code |
| entity_name | `entity_name` | String(200) | NOT NULL |
| account_code | `account_code` | String(50) | FK → accounts.account_code |
| account_name | `account_name` | String(100) | NOT NULL |
| summary | `summary` | String(500) | nullable |
| counterparty | `counterparty` | String(200) | nullable |
| amount_in | `amount_in` | Numeric(18,2) | ≥ 0, 与 amount_out 互斥 |
| amount_out | `amount_out` | Numeric(18,2) | ≥ 0, 与 amount_in 互斥 |
| rolling_balance | `rolling_balance` | Numeric(18,2) | nullable |
| state | `state` | String(20) | IN ('正常','待确认','异常','已作废') |
| source | `source` | String(20) | IN ('网银导入','手工录入','现金录入','票据录入','财务公司单据') |

## 关键索引

| 表 | 索引 | 用途 |
|----|------|------|
| `fund_events` | `idx_fund_events_date_account` | 报表按日期+账户查询 |
| `fund_events` | `idx_fund_events_state` | 按状态筛选（异常中心） |
| `fund_events` | `idx_fund_events_batch` | 按批次查询 |
| `parser_artifacts` | `uq_parser_artifacts_name_version` | 名称+版本唯一 |
| `parser_artifacts` | `idx_parser_artifacts_kind` | 按 kind+status 查找 active 解析器 |
| `report_templates` | `idx_report_templates_type` | 按 report_type 查找模板 |
